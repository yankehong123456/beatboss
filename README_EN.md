# 🕹️ Beat Boss: I'm So Good at It

A fun stress-relief HTML5 game where you can release work stress by clicking on a cartoon boss avatar!

## 🎮 Game Features

- **Simple & Fun**: Just click the boss avatar to play
- **30-Second Challenge**: Click as many times as possible within the time limit
- **Ranking System**: Earn different rank titles based on your click count
- **Rich Animations**: Visual feedback and smooth animations
- **Responsive Design**: Works on both desktop and mobile devices
- **Stress Relief**: Perfect for releasing work pressure through gameplay

## 🚀 How to Play

1. Open `index.html` in your web browser
2. Click the "Start Game" button
3. Click the boss avatar like crazy for 30 seconds
4. Check your final score and rank title

## 🏆 Ranking System

| Click Count | Rank Title | Description |
|-------------|------------|-------------|
| 0-49 | Lazy Intern 😐 | Need more effort |
| 50-99 | Cubicle Warrior 🙂 | Feeling better |
| 100-149 | Office Terminator 😎 | Solid smash |
| 150-199 | Stress Buster Pro 😤 | Excellent relief |
| 200+ | Boss Destroyer 🤯 | Maximum smash |

## 🎯 Game Controls

- **Mouse Click**: Click the boss avatar
- **Spacebar**: You can also use spacebar to hit (on game page)
- **Touch**: Mobile devices support touch controls

## 📱 Compatibility

- Modern browsers (Chrome, Firefox, Safari, Edge)
- Mobile device browsers
- PWA (Progressive Web App) support

## 🔧 Technology Stack

- HTML5
- CSS3 (animations and responsive design)

## 🎨 Customization

You can easily customize the game:

1. **Modify Boss Quotes**: Edit the `bossQuotes` array in `main.js`
2. **Adjust Ranking System**: Modify the `rankSystem` array
3. **Change Styles**: Edit the `style.css` file
4. **Add Sound Effects**: Add audio playback in the `hitBoss()` function

## 💰 Ad Integration

The game has pre-built ad slots for easy integration:
- Google AdSense (Web)
- Google AdMob (Mobile/PWA)

## 🚀 Deployment

1. Upload all files to a web server
2. Ensure HTTPS support (required for PWA)
3. Access `index.html` to start playing
